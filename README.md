# Phân tích trang web truyện tranh

1. Đối tượng sử dụng
- Quản trị viên
- Người dùng

2. Chức năng từng đối tượng

A. Quản trị viên
- Quản lí trang chủ, banner, quảng cáo
- Quản lí người dùng
- Quản lí truyện tranh

B. Người dùng
- Tìm kiếm truyện ( theo thể loại, tên truyện, tác giả )
- Bình luận 
- Rank

- Chức năng tìm kiếm
